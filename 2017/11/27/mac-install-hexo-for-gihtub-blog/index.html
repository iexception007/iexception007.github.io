mac install hexo for gihtub blog

安装组件

brew install git
brew install node

sudo npm install -g hexo
hexo init
hexo s
chrome open http://localhost:4000

npm install hexo-deployer-git –save
vim _config.yml
<<<<
deploy:
type: git
repository: https://github.com/iexception007/iexception007.gihtub.io.git
branch: master
<<<<

生成静态页面

hexo g

发布到github.com

hexo d

wait 10 minute.

发布新文件

hexo new “firstblog”
subl firstblog.md

hexo g
hexo d
